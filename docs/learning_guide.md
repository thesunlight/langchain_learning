# LangChain 学习指南

## 1. 项目概述

本项目旨在帮助学习 LangChain 框架的核心概念和实际应用。LangChain 是一个强大的框架，用于开发由语言模型驱动的应用程序。

## 2. 核心组件

### 2.1 模型 (Models)
- **LLMs**: 大型语言模型，如 GPT 系列
- **Chat Models**: 专为对话优化的模型
- **嵌入模型**: 用于将文本转换为向量表示

### 2.2 提示词 (Prompts)
- **提示词模板**: 动态生成结构化提示词
- **消息模板**: 为聊天模型构建消息序列
- **输出解析器**: 解析模型输出并结构化

### 2.3 链 (Chains)
- **LLMChain**: 将提示词模板和 LLM 结合
- **SequentialChain**: 按顺序执行多个链
- **RouterChain**: 根据输入动态选择处理路径

### 2.4 工具 (Tools)
- **自定义工具**: 用户定义的功能函数
- **预构建工具**: LangChain 提供的标准工具
- **工具集成**: 将工具与智能体结合使用

### 2.5 智能体 (Agents)
- **ReAct 智能体**: 使用推理和行动的智能体
- **工具使用**: 动态选择和使用工具
- **多工具协调**: 同时使用多个工具解决问题

### 2.6 记忆 (Memory)
- **对话记忆**: 在对话中保持上下文
- **缓冲记忆**: 存储和检索对话历史
- **总结记忆**: 自动生成对话总结

## 3. 学习路径

### 阶段 1: 基础概念
1. 理解 LLM 和聊天模型的使用
2. 学习提示词工程
3. 掌握基本链的构建

### 阶段 2: 中级应用
1. 创建自定义工具
2. 实现智能体
3. 使用记忆组件

### 阶段 3: 高级应用
1. 复杂链的构建
2. 多模态应用
3. 代理工作流

## 4. 实践建议

### 4.1 提示词工程最佳实践
- 明确指定任务和期望输出
- 提供示例以引导模型行为
- 使用分隔符区分不同部分
- 包含清晰的指令和约束

### 4.2 工具设计原则
- 单一职责：每个工具应专注于一个特定功能
- 输入验证：确保工具输入的安全性和有效性
- 错误处理：提供清晰的错误信息
- 文档说明：清楚描述工具的功能和用法

### 4.3 智能体设计考虑
- 工具选择：选择合适的工具组合
- 提示词设计：为智能体创建有效的提示词
- 性能优化：考虑延迟和成本因素

## 5. 项目结构说明

```
langchain_learning/
├── src/
│   ├── models/          # 模型使用示例
│   ├── prompts/         # 提示词工程示例
│   ├── chains/          # 链的构建和使用
│   ├── tools/           # 工具创建和集成
│   ├── agents/          # 智能体构建和应用
│   ├── memory/          # 记忆组件使用
│   └── utils/           # 实用工具函数
├── examples/            # 完整示例应用
├── docs/                # 详细文档和教程

├── .env                 # 环境变量配置
└── README.md            # 项目说明
```

## 6. 快速开始

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 配置环境变量：
   - 复制 `.env.example` 到 `.env`
   - 在 `.env` 中填入你的 API 密钥

3. 运行示例代码：
   ```bash
   python src/models/basic_models.py
   python src/prompts/prompt_engineering.py
   # ... 其他示例
   ```
